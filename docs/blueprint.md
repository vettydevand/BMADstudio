# **App Name**: BMAD IDE

## Core Features:

- Code Editor: A full-featured code editor with syntax highlighting, autocompletion, and real-time error detection tailored for BMAD method development.
- Integrated Terminal: An integrated terminal for executing BMAD commands and scripts directly within the IDE.
- BMAD Method Documentation: A searchable documentation browser that delivers up-to-date official information of BMAD, directly available inside the development environment.
- BMAD Code Generator Tool: Use LLM to parse plain text description and attempt to produce code using BMAD Method.

## Style Guidelines:

- Primary color: Dark indigo (#4B0082) to give a professional and sophisticated feel.
- Background color: Very dark gray (#222222) to reduce eye strain during long coding sessions.
- Accent color: Electric lime green (#32CD32) to highlight important interactive elements and commands.
- Body and headline font: 'Inter', a sans-serif font known for its legibility, is appropriate for UI elements and longer documentation.
- Code font: 'Source Code Pro' to clearly display BMAD code.
- Use a set of clear, minimalistic icons to represent different BMAD commands and functionalities.
- The layout is divided into resizable panels for the code editor, terminal, and documentation browser, allowing users to customize their workspace.
- Subtle animations are used to provide feedback when BMAD commands are executed.
